# Frontend

## ✏️ How to use

  ### 1. Cloning
  ```
  $ git clone https://github.com/Dayflt/Frontend.git
  ```

  ### 2. Install Packages & Run ⚛️
  - install packages
    ```
    $ cd Frontend
    $ npm install
    ```
  - run develop cilent
    ```
    $ npm start
    ```
    Checkout web service at http://localhost:3000
